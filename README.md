# React Memory Card Game

A **React-based memory matching game** built for learning and practice.

## Purpose
Created to practice:
- React hooks (`useState`, `useEffect`)
- Component-based design
- Game logic and state handling

## Inspiration
Inspired by work from another developer.
The project was implemented independently as part of my learning journey.

## Tech Stack
React • JavaScript • HTML • CSS 

## Features
- Shuffle & flip cards
- Match logic with score tracking
- Reset game functionality
